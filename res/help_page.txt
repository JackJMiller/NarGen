
Usage:
  new-world WORLD_NAME SAMPLE_PATH      create a new world according to a sample world
  generate CONFIG_FILE                  generate and save terrain world according to config file
  render WORLD_PATH                     render a saved world
  view WORLD_PATH                       view images of a world
  delete-terrain WORLD_PATH             delete generated terrain of a world
  list-sample-worlds                    list all sample world configurations
  -h, --help                            show this page
